4.解二元一次方程

  1. from sympy import *
  2. x=Symbol('x')
  3. y=Symbol('y')
  4. print solve([y+x-1,3*x+2*y-5],[x,y])

结果

 

注：(1)在计算之前要将变量设为符号

　　(2)要将变量一到一侧 y-1+x=0   3x+2y-5=0

6.求积分

 

  1. from sympy import *
  2. n = Symbol('n')
  3. s = ((n+3)/(n+2))**n
  4. print limit(s, n, oo)

　结果

 7.定积分

  1. from sympy import *
  2. t = Symbol('t')
  3. x = Symbol('x')
  4. m = integrate(sin(t)/(pi-t),(t,0,x))
  5. n = integrate(m,(x,0,pi))
  6. print n
